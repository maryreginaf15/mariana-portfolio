# Portfólio Profissional - Mariana Ferreira

Este repositório contém o código-fonte do meu portfólio profissional, focado em BI, Análise de Dados e Automação.

## 🚀 Tecnologias
- **HTML5**: Estrutura semântica e SEO.
- **CSS3**: Design responsivo com Vanilla CSS (Gradients, Glassmorphism, Flexbox/Grid).
- **JavaScript**: Animações de Scroll Reveal e interatividade.
- **Google Fonts & Font Awesome**: Tipografia e ícones.

## 📂 Estrutura do Projeto
- `index.html`: Página principal.
- `style.css`: Estilização completa e sistema de design.
- `script.js`: Lógica de interatividade e animações.
- `assets/`: Imagens, logos e currículo em PDF.

## 🔧 Como Executar Localmente
1. Clone o repositório.
2. Execute `npm install` (opcional, para ferramentas de desenvolvimento).
3. Abra o `index.html` em qualquer navegador ou use `npm start` se tiver o `live-server` instalado.
